---
title: useRouter
date: '2023-02-18'
---

React hook to use the router in a functional component (similar to the Router component in Next).
Contains all of the necessary funtions and objects for a basic router:<br />
`push` to push a new route to the stack<br />
`replace` to replace the current route with a new one<br />
`pathname` to get the current pathname<br />
`query` to get the current search and query params both in 1 object<br />
`search` to get the current search params alone<br />
`params` to get the current query params alone<br />
`match` declares the `useMatch` hook<br />
`location` declares the `useLocation` hook<br />
`history` to get the history of the stack<br />
